CTX GitHub 文件加速

GitHub文件链接带不带协议头都可以，支持release、archive以及文件，右键复制出来的链接都是符合标准的。

release、archive使用cf加速，文件会跳转至JsDelivr

注意，不支持项目文件夹

release源码：https://github.com/xxntmctx/project/archive/v0.1.0.tar.gz

release文件：https://github.com/xxntmctx/project/releases/download/v0.1.0/example.zip

分支文件：https://github.com/xxntmctx/project/blob/master/filename
